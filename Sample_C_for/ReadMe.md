## for 文の動作順序
```c
for (１;２;３)
	４
```
１→２→４→３→２→４→３→……
初めに１を１回だけ実行する。
続いて２を実行し、その結果が真であれば４、３を実行……これを繰り返す。
## コンマ演算子
```c
A, B
```
Aを評価しその結果は捨てる。次にBを評価し、その値を式全体の値として採用する。
```c
//例1
if (A, B) {
	//Bが真である場合の処理
}
else {
	//Bが偽である場合の処理
}
```
```c
//例2
while (A, B) {
	//Bが真である場合に続ける処理
}
```
### 評価順序
必ず A が先に評価され、その次に B が評価される。評価順序が逆になることはない。
### 他の演算子
評価順序が言語で規定される演算子は少ない。
&&, ||, ?: は左から右に評価される。
+, -, *, /, % などという演算子で左右どちらが先に評価されるかは決まっていない。
