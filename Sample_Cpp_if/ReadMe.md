## 値と真偽の基本原則
C言語と同じ原則であり、0は偽、0以外は真である。
演算子をオーバーロードすることでクラス・構造体・共用体の真偽を規定することができる。

### 偽
|記述|説明|
|---|---|
|0  |整数値の 0 は偽|
|.0|浮動小数点であっても 0 なら偽|
|*""|Null文字（文字コードは0）は偽|
|'\0'|C++ではchar型の0であり偽|
|nullptr|C++でnullptrは偽|
|メンバ変数の相対アドレス|0かどうかではなく、nullptrか非nullptrかが判断基準|

### 真

|記述|説明|
|---|---|
|12345  |非0の整数値は真|
|3/2.|浮動小数点であっても非0真|
|*"X"|'X'の文字コードは非0で真|
|""|文字列のアドレスはnullptrではなく真|
|sample_true|関数アドレスはnullptrではなく真|
|メンバ変数の相対アドレス|非nullptrであり真|
|演算子のオーバーロード|真偽は実装次第|
|explicit operator bool|新しいスタイル|
|operator const void*|古いスタイル|

## 関係演算子、比較演算子
### 演算結果の値
式が真であればtrue
式が偽であればfalse

### ポインタの大小関係
同一配列内の２要素を指すポインタ間で大小関係は意味を持つ。
そうではないデータや関数を指すポインタの間で大小を比較しても一般に無意味。

## 比較関数と _lfind, qsort
std::find は配列中の１要素を探す関数
std::sort は配列内の要素を並べ替える関数
### 比較関数
operator<によって大小関係を表す。
