## 値と真偽の基本原則
0は偽
0以外は真
### 偽
|記述|説明|
|---|---|
|0  |整数値の 0 は偽|
|.0|浮動小数点であっても 0 なら偽|
|*""|Null文字（文字コードは0）は偽|
|'\0'|C言語ではint型の0であり偽|
|NULL|C言語では0か(void*)0であり偽|

### 真

|記述|説明|
|---|---|
|12345  |非0の整数値は真|
|3/2.|浮動小数点であっても非0真|
|*"X"|'X'の文字コードは非0で真|
|""|文字列のアドレスはNULLではなく真|
|sample_true|関数アドレスはNULLではなく真|
## 関係演算子、比較演算子
### 演算結果の値
式が真であれば1
式が偽であれば0

### ポインタの大小関係
同一配列内の２要素を指すポインタ間で大小関係は意味を持つ。
そうではないデータや関数を指すポインタの間で大小を比較しても一般に無意味。

## 比較関数と _lfind, qsort
_lfind は配列中の１要素を探す関数
qsort は配列内の要素を並べ替える関数
### 比較関数
２要素が等しい場合は 0 を返す
左要素が右要素よりも大きい場合は正の値を返す
左要素が右要素よりも小さい場合は負の値を返す

左要素から右要素を差し引いた時の符号を考えると良い。
